# SMART — Annotation Field Reference

SMART (**S**omatic **M**utation **A**nnotation and **R**eporting **T**ool) integrates annotations
from multiple sources into a single output table. This page lists every field produced by the pipeline,
its source, version, and output tier.

<div class="warn">
⚠️ <strong>Tier assignments are configurable.</strong> The tiers shown here reflect the default
<code>Config.yaml</code> shipped with this release. Fields can be moved between tiers — or excluded
entirely — by editing <code>Config.yaml</code> and rebuilding the pipeline config. Do not rely on
tier numbers being fixed across different deployments.
</div>

## Output tiers

| Tier | File | Audience | Description |
|------|------|----------|-------------|
| **Tier 3** |  | Clinical scientists | Minimal, clinically focused fields |
| **Tier 2** |  | Bioinformaticians | Extended analytical fields |
| **Tier 1** |  | Downstream tools / MAF | All non-dropped fields |
| **Dropped** | *(excluded)* | — | Internal, redundant, or raw fields not included in any output |

The total number of columns in  varies between approximately **300 and 900**,
depending on how many OncoKB treatment, diagnostic, and prognostic implication entries are returned
for the variants in a given run (each entry is expanded into its own set of columns).

<script>
function filterTable() {
  const search = document.getElementById('search').value.toLowerCase();
  const src    = document.getElementById('src-filter').value;
  const tier   = document.getElementById('tier-filter').value;
  const rows   = document.querySelectorAll('#field-table tbody tr');
  let visible  = 0;
  rows.forEach(row => {
    const text = row.textContent.toLowerCase();
    const rowSrc  = row.dataset.source  || '';
    const rowTier = row.dataset.tier    || '';
    const matchSearch = !search || text.includes(search);
    const matchSrc    = !src    || rowSrc.toLowerCase().includes(src.toLowerCase());
    const matchTier   = !tier   || rowTier === tier;
    if (matchSearch && matchSrc && matchTier) {
      row.classList.remove('hidden');
      visible++;
    } else {
      row.classList.add('hidden');
    }
  });
  document.getElementById('result-count').textContent =
    visible + ' of ' + rows.length + ' fields shown';
}
filterTable();
</script>
